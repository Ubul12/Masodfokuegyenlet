Másodfokú Egyenlet Megoldása

A másodfokú egyenlet általános alakja:

ax² + bx + c = 0

A megoldóképlet:

x = (-b ± √(b² - 4ac)) / 2a

Hol használjuk?

A másodfokú egyenleteket a matematikában, fizikában, mérnöki tudományokban és gazdasági modellekben alkalmazzuk.

Mire használjuk?

Ezeket az egyenleteket mozgások, pályák, optimális értékek és egyéb természeti jelenségek kiszámítására használjuk.

Mikor használjuk?

Amikor olyan problémát oldunk meg, ahol egy mennyiség négyzetes kapcsolatban áll egy másik változóval.

Hogyan használjuk?

Behelyettesítjük az adott együtthatókat a megoldóképletbe, majd elvégezzük a számításokat a gyökök meghatározásához.

Python kód a másodfokú egyenlet megoldására:

import tkinter as tk
from tkinter import ttk, messagebox
import math

def solve_quadratic():
    try:
        a = float(entry_a.get())
        b = float(entry_b.get())
        c = float(entry_c.get())

        if a == 0:
            if b == 0:
                result = "Ez nem egy érvényes egyenlet." if c != 0 else "Minden x megoldás."
            else:
                result = f"Ez egy elsőfokú egyenlet. Megoldás: x = {-c / b}"
        else:
            szam1 = b**2 - 4*a*c
            if szam1 > 0:
                x1 = (-b + math.sqrt(szam1)) / (2 * a)
                x2 = (-b - math.sqrt(szam1)) / (2 * a)
                result = f"Két valós gyök van:\nx1 = {x1}\nx2 = {x2}"
            elif szam1 == 0:
                x = -b / (2 * a)
                result = f"Egyszeres gyök van: x = {x}"
            else:
                real_part = -b / (2 * a)
                imaginary_part = math.sqrt(-szam1) / (2 * a)
                x1 = complex(real_part, imaginary_part)
                x2 = complex(real_part, -imaginary_part)
                result = f"Két komplex gyök van:\nx1 = {x1}\nx2 = {x2}"
        messagebox.showinfo("Eredmény", result)
    except ValueError:
        messagebox.showerror("Hiba", "Kérlek, adj meg érvényes számokat!")
